The project will be implemented using \textbf{Python} as the primary programming language and \textbf{Jupyter Notebook} as the development environment for writing, testing, and documenting the code.

\begin{itemize}
    \item Data Manipulation and Analysis
    \begin{itemize}
        \item \textbf{pandas}: used for loading, cleaning, and working with data
        \item \textbf{numPy}: used for numerical calculations
    \end{itemize}
    \item Data Visualization
    \begin{itemize}
        \item \textbf{matplotlib}: used for basic plotting and visualization
    \end{itemize}
    \item Machine Learning and Modeling
    \begin{itemize}
        \item \textbf{scikit-learn}: used for model training, evaluation, and hyperparameter tuning
    \end{itemize}
\end{itemize}
